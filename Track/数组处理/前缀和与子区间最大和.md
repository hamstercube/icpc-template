前缀和求子区间和的最大

note: 该页的pre[i]代表下标为i的前缀和

1.求数组子区间最大
给你长度为n的数组求大小至少为1的子区间和

前缀和直接写(O(n))
```cpp
    ll v[1000100];
    ll pre[1000100], minn = 1e18, ans = -1e18; 
    for(ll i = 1; i <= n; i++){
        pre[i] = pre[i-1]+v[i];
        ans = max(ans, pre[i]-minn);
        minn = min(minn, pre[i]);
    }
```

2.在mod p的前提下求大小至少为1的子区间和

根据mod的性质与该页第一个track的原理可得，
答案为每pre[i]减去前面mod p 后最小的或者减去前面最接近pre[i]且比pre[i]大的
可以用set来写(O(nlog(n)))
```cpp
    set<ll> st;
    ll v[100010];
    ll pre[100010], ans = 0; 
    for(ll i = 1; i <= n; i++){
        pre[i] = (pre[i-1]+v[i])%p;
        if(s.size()){
            auto sm = st.begin();
            auto bi = st.upper_bound(pre[i]);
            if(sm != st.end()){ //似乎不用判断
                ans = max(ans, (pre[i]-*sm+p)%p);
            }
            if(bi != st.end()){
                ans = max(ans, (pre[i]-*bi+p)%p);
            }
        }
        st.insert(pre[i]);
    }
```
