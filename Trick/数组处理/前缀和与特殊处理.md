1.要求快速地求给定的数组a的l,r区间里每个元素ai*i的和（i为在区间的位置左端点从1开始计），例如l = 2, r = 5, a = {1, 2, 4 ,6 ,7}, 则答案为 2*1+4*2+6*3+7*4

思路：前缀和优化，一个前缀和prem存以l = 1，r = i的每个前缀的权值，再来一个前缀和pre存原来数组的前缀和, 可以用prem减去对应的前缀和pre的要求倍数来快速求权。

```cpp
for(ll i = 1; i <= n; i++){
    cin >> a[i];
    pre[i] = pre[i-1]+a[i];
    prem[i] = prem[i-1]+a[i]*i;
}
while(q--){
    cin >> l >> r;
    ll res = prem[r]-prem[l-1]-(l-1)*(pre[r]-pre[l-1]);
    cout << res << endl;
}
```
