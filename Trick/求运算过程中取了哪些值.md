求运算最过程中取了哪些值：

1.如背包dp问当使用最优放入背包的方法时有哪些物品要放入
    可以开个pre数组来承接前者

    设：dp[i] 为当背包已使用i空间的最优答案，
        v[i]为每个物品的价值，
        w[i]为每个物品的重量。

    则可以用以下方法来求最优方法中的其中一种取法
    if(dp[j] < dp[j-w[i]]+v[i]){  //需要注意 i>=w[i]
        dp[j] = dp[j-w[i]]+v[i];
        pre[j] = v[i];            //表示当背包为目前大小的最优放法时最后一个放入的物品价值
    }
    
    还原放入过程：
         vector<ll> ans;
         while(pre[wight]){
            ans.push_back(pre[wight]);
            wight -= pre[wight];
        }
