### 题意简化

**输入**：
- 长度为 $n$ 的排列 $p_1, p_2, \dots, p_n$
- 两个正整数 $x, y$（满足 $x, y < n$）

**允许操作**：
可进行任意次交换，每次可选下标 $i$，执行以下两种交换之一：
1.  交换 $p_i$ 与 $p_{i+x}$（要求 $i+x \le n$）
2.  交换 $p_i$ 与 $p_{i+y}$（要求 $i+y \le n$）

**问题**：
判断能否通过若干次操作（含0次），将排列 $p$ 变为升序排列 $[1, 2, \dots, n]$。

---
*名词解释：排列*
长度为 $n$ 的排列，是由 $1, 2, \dots, n$ 这 $n$ 个整数组成的、每个数恰好出现一次的数组。
例：$\{2, 3, 1, 5, 4\}$ 是长度为 5 的排列；$\{1, 2, 2\}$（含重复元素）、$\{1, 3, 4\}$（长度不足5）不是排列。


思路:
能相互交换的就放在一个连通块里,并查集实现

```cpp
const int N = 2e5 + 10;
 
vei v(N), a(N),fa(N);
 
int Find(int x){
    if(fa[x] == x) return x;
    return fa[x] = Find(fa[x]);
}
 
void solve(){
    int n, x, y;
    cin >> n >> x >> y;
    for(int i = 1; i <= n; i++){
        fa[i] = i;
        cin >> v[i];
        a[v[i]] = i;
    }
    for(int i = 1; i <= n; i++){
        if(i + x <= n){
            int f1 = Find(i), f2 = Find(i+x);
            if(f1 == f2) continue;
            fa[f2] = f1;
        }
        if(i + y <= n){
            int f1 = Find(i), f2 = Find(i+y);
            if(f1 == f2) continue;
            fa[f2] = f1;
        }
    }
    for(int i = 1; i <= n; i++){
        Find(i);
    }
    for(int i = 1; i <= n; i++){
        if(fa[a[i]] != fa[i]){
            cout << "No" << endl;
            return ;
        }
    }
    cout << "Yes" << endl;
}
```
